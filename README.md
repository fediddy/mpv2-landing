# MPV2 Landing Page

Static marketing site for [MPV2 Outreach](https://github.com/fediddy) — an automated cold-outreach pipeline that generates a bespoke website mockup for every prospect.

**Live:** deployed to GitHub Pages.

## What it demos

- The MPV2 5-stage pipeline (source → enrich → generate → send → replies)
- Live-embedded iframe of a real generated mockup (Woodall's Mechanical, Belmont CA)
- Stack + contact info

## Structure

- `index.html` — single-file marketing page (~18 KB)
- No build step, no framework, no JS deps

## Deploy

GitHub Pages via `main` branch, root path.
